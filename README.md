# 파이어 베이스 머신러닝
A collection of real life apps built using [Firebase ML Kit](https://firebase.google.com/products/ml-kit/) APIs.

1. 프로젝트 복사하기
2. 파이어베이스 콘솔에 추가하기
3. 앱 인스톨 하기
